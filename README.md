# telebot-ai-chatbot
A telegram ai chatbot using telegram and groq api


### Prerequisites
- Python 3.9
- A telegram account
- A Groq account for api key

### How to run code

- Create conda env

```bash
conda create -p venv python==3.9 -y
```

- Activate the env

```bash
conda activate venv
```

- Install required packages

```bash
pip install -r requirements.txt
```
